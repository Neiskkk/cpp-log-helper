## 2. Общая характеристика решения
Разработано расширение **C++ Log Helper**, предназначенное для автоматизации вставки строк вывода `std::cout` при отладке кода на языке C++.

Плагин вставляет строку вида:
```cpp
std::cout << "variable = " << variable << std::endl;
```
## 3. Пример работы

```cpp
int count = 5;
// После выделения count и нажатия Ctrl + Alt + N:
std::cout << "count = " << count << std::endl;
```
