Разработано расширение **C++ Log Helper**, предназначенное для автоматизации вставки строк вывода `std::cout` при отладке кода на языке C++.

Плагин вставляет строку вида:
```cpp
std::cout << "variable = " << variable << std::endl;
```
## Пример работы

```cpp
int count = 5;
// После выделения count и нажатия Ctrl + Alt + N:
std::cout << "count = " << count << std::endl;
```

## Архитектура

Плагин построен на стандартной архитектуре VS Code Extension API:

- **package.json**
  - регистрирует команду
  - задаёт сочетание клавиш
  - описывает метаданные расширения

- **extension.ts**
  - содержит реализацию логики
  - регистрирует команду
  - работает с текстом редактора
  - вставляет сгенерированный код

- **out/extension.js**
  - скомпилированная версия (генерируется TypeScript компилятором)

## Описание функций

### **activate(context)**  
Запускается при включении расширения.

- **Input:**  
  `context: vscode.ExtensionContext`  
- **Output:**  
  нет  
- **Назначение:**  
  регистрирует команду `cppLogHelper.insertCout`.

---

### **Команда cppLogHelper.insertCout**

Выполняется при нажатии **Ctrl + Alt + N**.

- **Input:**  
  - выделенный текст (имя переменной)  
  - либо пользовательский ввод (через InputBox)  
  - текущая позиция курсора  
- **Output:**  
  вставка строки:  
  ```cpp
  std::cout << "varName = " << varName << std::endl;
